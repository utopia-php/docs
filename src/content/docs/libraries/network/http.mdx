## Quick Start

### Installation

```bash
composer require utopia-php/http
```

### Basic Usage

```php
<?php

require_once __DIR__ . '/vendor/autoload.php';

use Utopia\Http\Http;
use Utopia\Http\Request;
use Utopia\Http\Response;

$http = new Http('0.0.0.0', 8080);

$http->get('/', function (Request $request, Response $response) {
    $response->json(['message' => 'Hello from Utopia HTTP!']);
});

$http->get('/users/{id}', function (Request $request, Response $response) {
    $id = $request->getParam('id');
    $response->json(['user_id' => $id]);
});

$http->start();
```

## API Reference

### Http Class

The main HTTP server class that handles incoming requests and manages routing.

#### Methods

- `__construct(string $host, int $port)` - Initialize the HTTP server
- `get(string $path, callable $callback)` - Register a GET route
- `post(string $path, callable $callback)` - Register a POST route
- `put(string $path, callable $callback)` - Register a PUT route
- `delete(string $path, callable $callback)` - Register a DELETE route
- `start()` - Start the HTTP server

### Request Class

Represents an incoming HTTP request with all its data and metadata.

#### Methods

- `getParam(string $key)` - Get route parameter value
- `getQuery(string $key)` - Get query parameter value
- `getHeader(string $key)` - Get request header value
- `getBody()` - Get request body content
- `getMethod()` - Get HTTP method
- `getPath()` - Get request path

### Response Class

Represents an outgoing HTTP response with various helper methods.

#### Methods

- `json(array $data, int $status = 200)` - Send JSON response
- `text(string $text, int $status = 200)` - Send text response
- `html(string $html, int $status = 200)` - Send HTML response
- `setHeader(string $key, string $value)` - Set response header
- `setStatus(int $status)` - Set HTTP status code

## Examples

### RESTful API

```php
$http->get('/api/users', function (Request $request, Response $response) {
    // Get all users
    $users = ['user1', 'user2', 'user3'];
    $response->json($users);
});

$http->post('/api/users', function (Request $request, Response $response) {
    // Create new user
    $data = json_decode($request->getBody(), true);
    // Process user creation...
    $response->json(['id' => 123, 'created' => true]);
});
```

### Middleware

```php
$http->middleware(function (Request $request, Response $response, callable $next) {
    // Log request
    error_log("Request: " . $request->getMethod() . " " . $request->getPath());
    $next($request, $response);
});
```

- **Fast Routing**: Optimized routing algorithm for quick request matching
- **Non-blocking I/O**: Support for async operations where possible
- **Zero Dependencies**: No external dependencies to slow down startup
