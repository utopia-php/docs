## Utopia PHP Libraries

*Micro PHP libraries for modern applications*

A comprehensive collection of micro PHP libraries designed for building high-performance microservice applications. Each library is focused, lightweight, and easy to use. The building blocks that help Appwrite serve billions of monthly requests and over 350k backends for web and mobile projects.
