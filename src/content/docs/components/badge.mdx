The Badge component is used to highlight and display short pieces of information like status, counts, or categories. It's designed to be compact and visually distinct.

## Usage

```jsx
<Badge variant="primary">New</Badge>
```

## Props

| Prop      | Type                                                            | Default     | Description               |
| --------- | --------------------------------------------------------------- | ----------- | ------------------------- |
| `variant` | `'primary' \| 'secondary' \| 'success' \| 'warning' \| 'error'` | `'primary'` | Visual style variant      |
| `size`    | `'sm' \| 'md' \| 'lg'`                                          | `'md'`      | Size of the badge         |
| `rounded` | `boolean`                                                       | `false`     | Whether to use pill shape |
| `icon`    | `ReactNode`                                                     | -           | Optional leading icon     |
| `onClick` | `function`                                                      | -           | Optional click handler    |

## Accessibility

- Uses appropriate color contrast ratios
- Interactive badges are keyboard accessible
- Icons have meaningful aria-labels
- Text content is readable at all sizes

## Best Practices

1. **Content Guidelines**

   - Keep text short and concise
   - Use clear and meaningful labels
   - Consider text truncation
   - Be consistent with variants

2. **Visual Design**

   - Maintain consistent spacing
   - Use appropriate color contrast
   - Consider icon alignment
   - Keep badges compact

3. **Usage Patterns**
   - Group related badges
   - Align with surrounding content
   - Consider mobile viewports
   - Use appropriate variants for context

## Examples

### Basic Variants

```jsx
<div className="flex gap-2">
	<Badge variant="primary">Primary</Badge>
	<Badge variant="secondary">Secondary</Badge>
	<Badge variant="success">Success</Badge>
	<Badge variant="warning">Warning</Badge>
	<Badge variant="error">Error</Badge>
</div>
```

### With Icons

```jsx
<div className="flex gap-2">
	<Badge variant="primary" icon={<StarIcon />}>
		Featured
	</Badge>
	<Badge variant="success" icon={<CheckIcon />}>
		Completed
	</Badge>
	<Badge variant="warning" icon={<AlertIcon />}>
		Pending
	</Badge>
</div>
```

### Sizes

```jsx
<div className="flex items-center gap-2">
	<Badge size="sm">Small</Badge>
	<Badge size="md">Medium</Badge>
	<Badge size="lg">Large</Badge>
</div>
```

### Rounded (Pill)

```jsx
<div className="flex gap-2">
	<Badge rounded>Default</Badge>
	<Badge rounded variant="primary">
		Primary
	</Badge>
	<Badge rounded variant="success">
		Success
	</Badge>
</div>
```

### Interactive

```jsx
<Badge variant="primary" onClick={() => console.log('Badge clicked')} style={{ cursor: 'pointer' }}>
	Click me
</Badge>
```

### With Counter

```jsx
<Badge variant="secondary">
	Messages
	<span className="badge-counter">5</span>
</Badge>
```
