The Action List component displays a vertical list of interactive items that trigger actions when selected. It's commonly used in menus, dropdowns, and navigation.

## Usage

```jsx
<ActionList>
	<ActionListItem onClick={() => console.log('Edit')}>
		<EditIcon />
		Edit
	</ActionListItem>
	<ActionListItem onClick={() => console.log('Delete')}>
		<DeleteIcon />
		Delete
	</ActionListItem>
</ActionList>
```

## Props

### ActionList

| Prop      | Type                     | Default     | Description                      |
| --------- | ------------------------ | ----------- | -------------------------------- |
| `variant` | `'default' \| 'compact'` | `'default'` | Controls the density of the list |
| `size`    | `'sm' \| 'md' \| 'lg'`   | `'md'`      | Controls the size of list items  |

### ActionListItem

| Prop       | Type        | Default | Description                          |
| ---------- | ----------- | ------- | ------------------------------------ |
| `onClick`  | `function`  | -       | Handler called when item is clicked  |
| `disabled` | `boolean`   | `false` | When true, prevents user interaction |
| `selected` | `boolean`   | `false` | Indicates if item is selected        |
| `icon`     | `ReactNode` | -       | Optional leading icon                |

## Accessibility

- Keyboard navigation (Arrow keys, Enter)
- ARIA roles and attributes for list structure
- Focus management and visible focus indicators
- Screen reader announcements for selection state

## Best Practices

1. **Content Structure**

   - Keep items short and actionable
   - Use consistent icon placement
   - Group related actions together

2. **Visual Hierarchy**

   - Use icons to enhance recognition
   - Maintain consistent spacing
   - Provide clear hover and active states

3. **Interaction Design**
   - Ensure adequate touch targets
   - Provide immediate feedback
   - Consider keyboard navigation

## Examples

### With Icons

```jsx
<ActionList>
	<ActionListItem icon={<EditIcon />} onClick={() => {}}>
		Edit file
	</ActionListItem>
	<ActionListItem icon={<ShareIcon />} onClick={() => {}}>
		Share file
	</ActionListItem>
	<ActionListItem icon={<DeleteIcon />} onClick={() => {}}>
		Delete file
	</ActionListItem>
</ActionList>
```

### With Selection

```jsx
<ActionList>
	<ActionListItem selected onClick={() => {}}>
		Option 1
	</ActionListItem>
	<ActionListItem onClick={() => {}}>Option 2</ActionListItem>
	<ActionListItem disabled onClick={() => {}}>
		Option 3 (Disabled)
	</ActionListItem>

```
