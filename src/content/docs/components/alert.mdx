The Alert component is used to communicate important messages, feedback, or status information to users. It supports different variants to convey different types of information.

## Usage

```jsx
<Alert variant="info">
	<AlertTitle>Update Available</AlertTitle>
	<AlertDescription>
		A new software version is available. Please update to access the latest features.
	</AlertDescription>
</Alert>
```

## Props

### Alert

| Prop      | Type                                          | Default  | Description                      |
| --------- | --------------------------------------------- | -------- | -------------------------------- |
| `variant` | `'info' \| 'success' \| 'warning' \| 'error'` | `'info'` | The style variant of the alert   |
| `icon`    | `ReactNode`                                   | -        | Optional custom icon             |
| `onClose` | `function`                                    | -        | Handler for dismissing the alert |

### AlertTitle

| Prop | Type     | Default | Description               |
| ---- | -------- | ------- | ------------------------- |
| `as` | `string` | `'h5'`  | HTML element to render as |

### AlertDescription

| Prop | Type     | Default | Description               |
| ---- | -------- | ------- | ------------------------- |
| `as` | `string` | `'div'` | HTML element to render as |

## Accessibility

- Uses appropriate ARIA roles and attributes
- Color combinations meet WCAG contrast requirements
- Icons have proper aria-labels
- Dismissible alerts are keyboard accessible

## Best Practices

1. **Content Guidelines**

   - Keep messages clear and concise
   - Use appropriate variant for context
   - Include actionable information
   - Avoid multiple alerts when possible

2. **Visual Design**

   - Use consistent spacing
   - Ensure sufficient contrast
   - Consider icon placement
   - Maintain hierarchy with other elements

3. **Interaction**
   - Make dismissal obvious
   - Consider auto-dismissal timing
   - Provide feedback on actions
   - Handle stacked alerts appropriately

## Examples

### Basic Variants

```jsx
<Alert variant="info">
  This is an informational message.
</Alert>

<Alert variant="success">
  Your changes have been saved successfully.
</Alert>

<Alert variant="warning">
  Your session will expire in 5 minutes.
</Alert>

<Alert variant="error">
  An error occurred while processing your request.
</Alert>
```

### With Custom Icon

```jsx
<Alert variant="info" icon={<CustomIcon />}>
	<AlertTitle>Custom Alert</AlertTitle>
	<AlertDescription>This alert uses a custom icon instead of the default one.</AlertDescription>
</Alert>
```

### Dismissible Alert

```jsx
<Alert variant="success" onClose={() => console.log('Alert dismissed')}>
	<AlertTitle>Success!</AlertTitle>
	<AlertDescription>
		Your profile has been updated. Click the X to dismiss this message.
	</AlertDescription>
</Alert>
```
