## Usage Guidelines

### Margins

- Use consistent margins between sections
- Apply larger margins for major content blocks
- Use smaller margins for related content

### Padding

- Use consistent padding within components
- Increase padding for larger screen sizes
- Maintain sufficient touch targets on mobile

### Layout

- Use grid and flex layouts with consistent gaps
- Apply responsive spacing based on viewport
- Maintain visual hierarchy with spacing
- Use whitespace effectively to improve readability
